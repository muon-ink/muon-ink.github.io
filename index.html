This project explores the concept that a digital monitor is one of a countless number of amazing and wonderful types of artistic media. The idea is to create an artistic space where art can be shared with others in a comfortable, relaxed manner.

The artworks below are released under a Creative Commons Attribution-NonCommercial-NoDerivatives 4.0 International (CC BY-NC-ND 4.0) license.

The artist's "name", Muon Bell, is a pseudonymn - this can sometimes enhance creativity.

This particular art is part of a larger project using a hybrid approach - film cameras and lenses, natural light (outdoor, no flash), slide film, digital scans, and a huge variety of amazing open-source image editing software to have some fun with it. This artwork is created almost entirely with open-source software. Thanks, Ubuntu Studio!

We're experimenting with using OpenTimestamps to "anchor" the image file to the Bitcoin blockchain. OpenTimestamps is an open-source project. It scales, so it can be very eco-friendly. There's a JavaScript client available on their Github repository.
